Before you start

Four things. Three are stuff you already have.

  • A laptop running Windows 10/11. Mac works, examples are Windows.
  • GitHub Copilot access via your Microsoft enterprise account. Check at github.com/settings/copilot.
  • Two GitHub accounts wired up: your personal one + your *_microsoft EMU account. The Setup, the lazy way lesson walks you through this.
  • Optional but recommended: an Azure subscription you can az login to.

No coding experience needed for the first five lessons. If you can install software and follow numbered steps, you're set.

How this differs from the GitHub original

The GitHub original is excellent but built for software developers working on a Python book app. This version:

  • Uses partner work scenarios throughout: RFPs, briefings, vTeam updates, PMX projects.
  • Adds the EMU/personal account dance that's specific to Microsoft.
  • Adds MCP servers we actually use (PMX, M365, GitHub) instead of generic database examples.
  • Calls out data boundaries (NDA content, partner data) explicitly.
  • Splits into core + optional so non-coders can stop at the value plateau.

The source course is referenced where it goes deeper than we need to.
